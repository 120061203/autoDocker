AutoDocker 測試頁面

這個頁面用於測試文件上傳功能

測試步驟：

  1. 打開瀏覽器開發者工具 (F12)
  2. 訪問 http://localhost:3001
  3. 上傳示例文件或輸入 GitHub 連結
  4. 點擊 "開始分析"
  5. 查看控制台輸出

示例文件：

您可以使用 examples/ 目錄中的示例文件進行測試：

  • Node.js 示例: examples/nodejs-app/package.json + examples/nodejs-app/index.js
  • Python 示例: examples/python-app/requirements.txt + examples/python-app/app.py

GitHub 連結測試：

在 GitHub 連結輸入框中輸入：

https://github.com/facebook/react

然後點擊 "分析" 按鈕
